# 你好，世界

欢迎来到我的个人博客。这个空间将记录我的思考、学习笔记以及对 AI 领域的探索。

### 为什么建立这个博客？
我一直相信写作是最好的思考方式。通过将琐碎的想法结构化，我能更清晰地认识到自己的成长路径。

### 技术架构
- **样式**: 基于现代 Bento Grid 风格。
- **构建**: Python 自定义脚本，自动将 Markdown 转换为 HTML。
- **发布**: GitHub Actions 自动部署到 GitHub Pages。
